 # FealtyX - GoLang REST API for Student Management

## Overview
This project demonstrates the creation of a simple REST API that manages student data (ID, Name, Age, Email). The API performs basic CRUD (Create, Read, Update, Delete) operations and integrates with the Ollama AI platform to generate summaries of student profiles.

## API Endpoints
The API exposes the following endpoints:

### 1. Create a New Student
- **Method:** POST
- **Endpoint:** `/students`
- **Request Body:**
    ```json
    {
        "name": "John Doe",
        "age": 21,
        "email": "john@example.com"
    }
    ```
- **Response:**
    ```json
    {
        "id": 1,
        "name": "John Doe",
        "age": 21,
        "email": "john@example.com"
    }
    ```

### 2. Get All Students
- **Method:** GET
- **Endpoint:** `/students`
- **Response:**
    ```json
    [
        {
            "id": 1,
            "name": "John Doe",
            "age": 21,
            "email": "john@example.com"
        },
        {
            "id": 2,
            "name": "Jane Smith",
            "age": 22,
            "email": "jane@example.com"
        }
    ]
    ```

### 3. Get a Student by ID
- **Method:** GET
- **Endpoint:** `/students/{id}`
- **Response:**
    ```json
    {
        "id": 1,
        "name": "John Doe",
        "age": 21,
        "email": "john@example.com"
    }
    ```

### 4. Update a Student by ID
- **Method:** PUT
- **Endpoint:** `/students/{id}`
- **Request Body:**
    ```json
    {
        "name": "Johnathan Doe",
        "age": 22,
        "email": "johnathan@example.com"
    }
    ```

### 5. Delete a Student by ID
- **Method:** DELETE
- **Endpoint:** `/students/{id}`
- **Response:** `Student deleted successfully`

### 6. Generate a Summary of a Student by ID using Ollama
- **Method:** GET
- **Endpoint:** `/students/{id}/summary`
- **Response:** A summary generated by Ollama AI, describing the student's profile.

## Ollama Integration
To integrate Ollama for generating AI-based summaries:
1. Install Ollama on your localhost by following the [quickstart guide](https://ollama.com/quickstart).
2. Install the Llama3 language model for Ollama.
3. Make API requests to Ollama from your Go application. Pass the student object as input to generate a summary.

### Example Request to Ollama API:
```go
url = "http://localhost:11434/api/generate"
response, err := http.Post(response = requests.post(url, headers=headers, data=json.dumps(data)))

```
